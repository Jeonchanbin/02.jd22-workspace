[ PC에 경로설정 ]
설정 > 시스템 > 정보 > 고급 시스템 설정
-> 창에서 [환경변수...] 클릭

환경벼수란 -> 프로그램 실행시 전역적으로 경로를 등록하여 잡아주는 곳 

"JAVA_HOME" 이름으로 jdk11 버전 폴더까지 잡아 등록
path에 "%JAVA_HOME%\bin"을 등록
앞의 경로가 "JAVA_HOME"에 담겨 있고 %(퍼센트)사이에 이걸 씀
bin폴더까지 잡아주면 자바실행 파일이 전역적으로 등록되어 어디서나 호출가능

[ 등록후 확인 ]
1. cmd 창 실행(실행에 cmd 엔터)
2. java -version 확인 -> 버전이 나옴
3. javac -varsion 확인 -> 버전이 나옴
-> 확인 안될 경우 재무팅 환경변수 등록여부 재확인
